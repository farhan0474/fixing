Steps to install the project:

1. Download the latest commit from gitlab link: GitLab project
2. Open the folder project -> database and download the database file
3. Log in to the MyWeb Server's phpMyAdmin by going to this link
4. Log in with your credentials and press on the Import button
5. Select the sql file downloaded from the project's database folder
6. Then upload the whole project folder to MyWeb Server's public html
7. Then run the page: myuser.myweb.cs.uwindsor.ca/project/test/html/test.php where myuser is your username
